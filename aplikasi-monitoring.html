Aplikasi Mobile Monitoring Detak Jantung, Saturasi Oksigen, dan Suhu Tubuh Menggunakan Flutter dan Esp32

Source Code Esp32 Source Code Aplikasi Mobile Demo Program

Deskripsi

Proyek ini melibatkan pengembangan aplikasi mobile untuk monitoring kesehatan menggunakan bahasa pemrograman Dart dan framework Flutter, serta integrasi dua sensor kesehatan: MAX30102 untuk mengukur kadar oksigen darah dan detak jantung, dan MLX90614 untuk mengukur suhu tubuh. Sensor-sensor ini terhubung melalui mikrokontroler ESP32 yang menggunakan protokol komunikasi Bluetooth Low Energy (BLE) untuk mengirimkan data ke aplikasi mobile. Untuk database, proyek ini menggunakan Firebase Firestore.

Sistem ini dirancang untuk memberikan cara yang efisien dan praktis bagi pengguna dalam memantau kesehatan mereka secara real-time, di mana saja, serta menyimpan data ke server untuk referensi di masa depan.

Fitur

Berikut adalah penjelasan lengkap mengenai beberapa fitur utama yang diimplementasikan:

  1. Real-time Health Monitoring: Aplikasi menampilkan data kesehatan secara langsung dari sensor melalui koneksi Bluetooth Low Energy (BLE), seperti kadar oksigen dalam darah, detak jantung, dan suhu tubuh.
  2. Data Storage: Data kesehatan yang diambil dari sensor disimpan di server, sehingga pengguna dapat mengakses riwayat pemeriksaan mereka kapan saja.
  3. User-friendly Interface: Aplikasi dirancang dengan antarmuka yang mudah digunakan, memudahkan pengguna untuk melakukan pengecekan kesehatan secara mandiri kapan saja dan di mana saja.
  4. Low Power Consumption: Penggunaan Bluetooth Low Energy (BLE) menjamin konsumsi daya yang rendah, memungkinkan perangkat digunakan dalam waktu yang lama tanpa sering mengisi ulang baterai.
